# Linux Fundamentals (TryHackMe Series)

> Hands-on Linux training covering command-line usage, system operations, and foundational cybersecurity concepts.

---

## 📌 Overview

This project documents my completion of the Linux Fundamentals series on TryHackMe.
It demonstrates practical experience using Linux in a controlled lab environment, focusing on skills directly applicable to IT support and cybersecurity roles.

The series progresses from basic command-line interaction to system-level operations such as process management, automation, and log analysis.

---

## 🧠 Skills Gained

* Linux command-line navigation (`ls`, `cd`, `pwd`, `cat`)
* File and directory manipulation (`touch`, `mv`, `cp`, `rm`)
* Searching and filtering data (`grep`)
* Remote access using SSH
* User and permission management
* Process and service control (`ps`, `kill`, `systemctl`)
* Task automation with cron jobs
* Log analysis (`/var/log`, Apache logs)
* File transfer and hosting (`wget`, Python HTTP server)
* Basic shell operators and command chaining

## 🔍 Key Concepts Demonstrated

### 1. Command-Line Proficiency

Used essential Linux commands to navigate the filesystem, manage files, and interact with the system.

### 2. System Interaction

### 3. File & Data Handling

Created, moved, and analyzed files, including searching logs using `grep`.

### 4. Process & Service Management

Identified running processes, terminated them, and managed services using `systemctl`.

### 5. Automation
